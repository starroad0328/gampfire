## 내용

각 일지에는 다음 정보가 포함됩니다:

- **완료한 작업**: 해당 날짜에 구현한 기능 목록
- **기술 스택**: 사용한 기술 및 라이브러리
- **기술 결정**: 중요한 기술적 결정 사항
- **개선 효과**: UX/성능 개선 사항
- **참고사항**: 추가 메모 및 향후 고려사항
