### 12. API 라우트 동적 렌더링 오류 수정

**문제:**
- Next.js 빌드 시 "Dynamic server usage" 오류 발생
- API 라우트가 `searchParams`와 `headers`를 사용하는데 정적 렌더링 시도

**해결 방법:**
- 5개 API 라우트에 `export const dynamic = 'force-dynamic'` 추가
- `revalidate` 설정과 충돌하여 모든 `revalidate` 제거

**수정 파일:**
- `src/app/api/games/list/route.ts`
- `src/app/api/games/hot/route.ts`
- `src/app/api/games/recommended/route.ts`
- `src/app/api/games/search/route.ts`
- `src/app/api/reviews/list/route.ts`

**커밋:**
- 8b018c3 - Fix dynamic server usage errors by adding force-dynamic export
- 20d7073 - Remove revalidate config to fix dynamic rendering conflicts
