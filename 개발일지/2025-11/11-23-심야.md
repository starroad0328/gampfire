# 개발일지 - 2025년 11월 23일 심야

## 완료한 작업

### Steam 태그 기반 추천 시스템 구현
- Steam Spy API 연동 (getSteamTags 함수)
- Game 모델에 tags 필드 추가
- 게임 조회 시 Steam 태그 자동 수집 및 저장
- 추천 알고리즘 개선
  - 태그 2배 가중치 적용
  - 장르 + 태그 매칭 조합
- DB 우선 검색 → IGDB 보충 방식 채택
- 비로그인 사용자 추천 탭 숨김 처리 (401 에러 수정)

### 온보딩 페이지 로직 개선
- 신규 사용자: 최소 10개 평가 필수 (기존 동일)
- 재방문 사용자 (10개 이상 평가 완료): 갯수 제한 없이 자유 평가
- `/api/user/review-count` 엔드포인트 추가
- UI 메시지 및 카운터 표시 조건부 처리
