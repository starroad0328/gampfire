# 개발일지 - 2025년 11월 19일

## 완료한 작업

### 리뷰 시스템 UI/UX 개선 및 색상 테마 변경

#### 대표색 변경: 파랑색 → 주황색
- globals.css의 primary 색상 변경 (HSL: 25 95% 53%)
- 버튼, 링크, 포커스 링 등 모든 primary 색상 적용
- Steam 페이지 가기 버튼 주황색 적용
- 리뷰 추천 버튼 주황색 적용
- 트레일러 재생 버튼 주황색 적용

#### 평점 통계 색상 구분
- 인증된 평가 (전문가/인플루언서): 파란색 별 (fill-blue-500)
- 일반 평가 (일반 유저): 주황색 별 (fill-orange-500)

#### 리뷰 별점 색상 역할 기반 구분
- 전문가/인플루언서 리뷰: 파란색 별
- 일반 유저 리뷰: 주황색 별
- 게임 상세 페이지 및 전체 리뷰 페이지 모두 적용

#### 추천/비추천/보통 기준 변경
- 추천: 3.0점 초과 (파란색 배지)
- 보통: 2.5~3.0점 (회색 배지)
- 비추천: 2.5점 미만 (빨간색 배지)

#### 리뷰 배지 디자인 개선
- 배지를 리뷰 카드 최상단으로 이동 (프로필 위)
- 배경색, 테두리, 패딩 추가로 가독성 향상
- 아이콘 + 텍스트 조합 (ThumbsUp, Minus, ThumbsDown)

#### 게임 상세 페이지 리뷰 섹션 재구성
- 기존 2:1 → 새로운 2:1:1 레이아웃
  - 추천 평가 (2칸): 좋아요가 많은 리뷰
  - 인증된 평가 (1칸): 전문가/인플루언서 리뷰만 필터링
  - 최신 평가 (1칸): 최근 작성된 리뷰

#### 리뷰 줄 바꾸기 저장 문제 수정
- 모든 리뷰 코멘트에 whitespace-pre-wrap 클래스 추가
- 게임 상세, 전체 리뷰, 프로필 페이지 모두 적용
