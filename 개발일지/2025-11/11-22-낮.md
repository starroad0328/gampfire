# 개발일지 - 2025년 11월 22일 낮

## 완료한 작업

### 커뮤니티(동아리) 시스템 기본 구조 완성
- 동아리 목록 페이지 (`/communities`)
- 동아리 생성 페이지 및 API
- 동아리 상세 페이지
  - 게시글 목록
  - 멤버 목록
- 동아리 가입/탈퇴 기능
- 동아리장 왕관 뱃지 표시
- 동아리 관리 페이지 (동아리장 전용)

### 게시판 시스템 구현
- Board 모델 추가 (Prisma 스키마)
- 게시판 생성/수정/삭제 API (동아리장 전용)
- 게시판별 게시글 필터링
- 게시판 관리 UI (BoardManagement 컴포넌트)
- 카테고리 기능 (게시판 그룹화)

### 공지사항 시스템
- isNoticeBoard 필드 추가 (공지사항 전용 게시판)
- 공지사항 게시판은 동아리장만 글 작성 가능
- 공지사항 게시판에 작성된 글 자동으로 공지 처리
- 공지사항 시각적 표시
  - 빨간색 라벨 및 배경
  - 모든 게시판에서 상단 고정

### 게시글 기능
- 게시글 작성/수정/삭제 기능
- 게시글 상세 페이지
- 이미지 업로드 기능
  - 다중 이미지 지원
  - 최대 5MB
- 이미지 미리보기 및 삭제
- 이미지 있는 게시글 아이콘 표시
- 더보기 메뉴 (수정/삭제)
- 댓글 시스템 (작성/삭제)
- 좋아요 기능

### UI/UX 개선
- 동아리 카드 디자인
- 게시판 태그 표시
- 공지사항 빨간색 강조
- 왕관 뱃지 (동아리장 표시)
- 이미지 그리드 레이아웃

## 기술 스택
- Prisma (Board, Post, Comment 모델)
- Next.js (동적 라우팅)
- 이미지 업로드 (파일 시스템)

## 데이터 모델
- Community: 동아리
- CommunityMember: 동아리 멤버
- Category: 카테고리
- Board: 게시판
- Post: 게시글
- Comment: 댓글
- PostLike: 게시글 좋아요
